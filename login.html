MyRed requires a NU ID login.

Passwords are set using TrueYou. TrueYou is a system that helps you maintain a single password and NU ID across many systems at many institutions in Nebraska.

Participating institutions include: UNL, UNK, UNO, UNMC, Chadron State College, Peru State College, and Wayne State College.

If you are an applicant to one or more of the institutions above, your NU ID and password for each campus will be the same.

To use MyRED, please use one of the following options:

  • Add myred.nebraska.edu as a trusted site in your browser. In Internet Explorer, go to Tools > Internet Options > Privacy. Click Sites then type "myred.nebraska.edu" and click Allow.
  • Change your privacy settings to medium or low to allow MyRED site to write cookies to your computer. In Safari, go to Settings > Preferences > Security. For "Accept Cookies:" select Always.
